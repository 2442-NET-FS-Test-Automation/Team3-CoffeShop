# Coffee Shop Management System 

A role-based coffee shop management system designed to simplify inventory management and order processing for managers and baristas.

## Features

### Manager

Managers have full control over the inventory and can:

* Add new products to the inventory.
* Update product information.
* Increase stock quantities.
* View inventory reports and product information.

### Barista

Baristas have limited permissions and can:

* Decrease product quantities when ingredients are used.
* View available inventory.
* Create and deliver customer orders.
* Mark orders as completed.

## User Roles

| Role    | Permissions                                                |
| ------- | ---------------------------------------------------------- |
| Manager | Add products, manage inventory, update stock, view reports |
| Barista | Consume inventory items and deliver orders                 |

## Technologies Used

* **Backend:** ASP.NET Core Web API
* **Database:** SQL Server
* **ORM:** Entity Framework Core
* **Authentication:** JWT Authentication
* **Documentation:** Swagger / OpenAPI
